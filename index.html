Comandos do Git

Configuração

        
            git config --global user.name "Seu Nome"
            git config --global user.email "seuemail@example.com"

Repositório

        
            git init
            git clone 
            git remote add origin

Branches

        
            git branch
            git branch 
            git checkout 
            git merge 
            git branch -d

Commit

        
            git status
            git add 
            git commit -m "Mensagem do commit"
            git push origin

Atualização

        
            git pull origin 
            git fetch
            git merge origin/

Outros Comandos

        
            git log
            git diff
            git reset
            git revert
